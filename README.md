# Sleek

# to run backend service locally
`node index.js` and go to `http://localhost:3000/`

go to `http://localhost:3000/deals/all` to fetch all data or go to `http://localhost:3000/deals/<start-range>/<end-range>` to get the data from the range [start, end)

example: `http://localhost:3000/deals/0/10` will return all the data from indexes 0, 1, 2, ..., 9.
